## 动机

EXP-076 发现 orth_only (L_orth=1.0, lr=0.1) 在 R200 时 AVG Last 73.87（s=333 崩到 65.34%），严重不稳。测试：
1. L_orth 权重扫：0.5 / 1.0 / 2.0
2. HSIC 非线性独立：lh=0.1
3. LR 扫：0.1 → 0.05

## 变体通俗解释

- **lo=0.5/2.0**: 正交约束减半/加倍
- **+HSIC**: 余弦正交 + Hilbert-Schmidt 独立性损失
- **LR=0.05**: baseline 是 0.1，减半 LR 看是否让后期收敛更稳（因为 decay=0.9998）

## 🏆 关键发现

1. **LR=0.05 是今日最大突破** — PACS 严格同 seed {2,15,333} 下四个指标（ALL/AVG × Best/Last）全部超 FDSE
2. **drop AVG 从 baseline 2.59 → 0.99** — 稳定性大幅改善
3. **lo=0.5 / lo=2.0 / HSIC 均不超过 lo=1.0** — L_orth=1.0 接近最优；高 peak 但 drop 也大
4. **Office AVG Last 仅差 FDSE 0.51%** — 接近打平
5. **因果**：baseline LR=0.1 + decay 0.9998 → R200 时 LR 仍约 0.067，过大；LR=0.05 → R200 时 LR 约 0.034，后期稳定收敛
