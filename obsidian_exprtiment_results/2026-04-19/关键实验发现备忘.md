## 发现 2: flgo 的 AVG Best vs ALL Best 命名混淆

**背景**:之前的 EXP-080/083/084 NOTE 里把 `mean_local_test_accuracy`(sample-weighted 即 ALL)误称为 "AVG Best"。

**事实(从 JSON schema 验证)**:
- `mean_local_test_accuracy` = sample-weighted mean(各 client 按 test set 大小加权)= **ALL**
- `local_test_accuracy` = client-equal mean(每 client 等权)= **AVG**
- `local_test_accuracy_dist` = per-client list(per-domain 数据源)

**之前 NOTE 里 "AVG Best 80.41 / 88.61 / 89.82" 实际都是 ALL Best**

**为什么以前没出问题**:论文会更关注 AVG(equal-weight across domains),但之前几个 NOTE 都只放一个数字,未明确区分。现在 EXP-095 补齐了两个指标,发现差距极大(Office Plan A AVG 82.55 vs ALL 88.61 差 6%),暴露混淆。

**后续实验都要两个指标都 report**。
