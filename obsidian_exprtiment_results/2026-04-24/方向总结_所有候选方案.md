## 一、今日 (2026-04-24) 的 5 大关键发现

### 发现 1: EXP-125 OCSD 验证 1 推翻假设 (早晨 ~04:00)

- 对 24 张 over_conf_wrong 图人眼审阅
- 用户: "图不极端, 人眼能认, model 学歪了"
- 结论: **D 类** (model class boundary 系统性错位), 不是 A 类 (style shortcut)
- OCSD/PCH 方向都失败

### 发现 2: Decouple Probe 颠覆"shared-trunk 诅咒"假说 (下午 ~05:00)

- 3 个 ckpt (PACS s=2/s=15 VIB + Office s=15 sgpa) 测 W_sem·W_sty^T, channel usage
- **L_orth 在权重 + channel 层面已近乎完美** (cos 均值 0.025, channel pearson ~0)
- 但 EXP-111 probe_sty_class MLP-256 = **0.81** 泄漏仍在
- **新机制**: L_CE 让 trunk 每个 channel 都含 class 信息 → z_sty 总能探出
- **病根在 trunk class-wide dispersion, 不在 head 共享**
- **三选项判决**:
  - A 权重行正交: 无效 (已经在物理下限)
  - B channel mask 互斥: 无效 (channel 已自然独立)
  - **C spatial mask (F2DC)**: 唯一 head 路线存活

### 发现 3: PGA 方法论死亡 (晚 ~07:00-08:00)

- 3 版 pixel-to-prototype attribution 都失败 (Photo/Sketch 反向: 红在背景蓝在主体)
- **数学病根**: cos-with-mean-prototype 平均操作强化"多样本一致部分"(背景), 抹平"差异大部分"(主体)
- 合照类比: 100 学生合影叠加, 墙清晰、人糊 → 新图比对时墙位置 cos 高
- **用户洞察**: "像素去逼近原型不可取, 要像 F2DC **训一个东西**学判别特征"
- **正式关闭** PGA 方向: 要救就回归 Grad-CAM, novelty 归零

### 发现 4: FedDSA-Calibrator 兜底方案成文 (晚 ~08:30)

- 位置: pooled 级加 50 行 corrector MLP, 从 z_sty 显式提 class 信号
- Loss: L_CE + L_orth + λ_aux · CE(aux_cls(z_sty_cal), y)
- Final: classifier(z_sem + λ · z_sty_cal)
- 定位: **兜底 A, 不是主攻** (Office 目标 +0.4~2.5pp, 1 新超参, 不治本)

### 发现 5: DualEncoder 才是真·兜底 (晚 ~09:00)

- 用户纠正: "兜底是走双通道训练, 参数量变大那个"
- 架构: **两个完整 AlexNet encoder** (encoder_sem + encoder_sty)
- 核心: L_CE **只反传 encoder_sem**, encoder_sty 只被 L_orth/L_sty_aux 训
- **物理保证**: encoder_sty trunk 不含 class 梯度历史 → class mirror 泄漏源头杜绝
- **Probe 证据**: probe_sty_class 0.81 → <0.3 预期
- 参数量 2×, 通信 1× (encoder_sty 本地私有)
- **新兜底排序**: DualEncoder 首选兜底 > Calibrator 次选兜底
