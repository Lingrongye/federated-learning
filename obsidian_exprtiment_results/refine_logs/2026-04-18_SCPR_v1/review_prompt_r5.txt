## Method Thesis

## Proposed Method

### Notation

- s_k := normalize(E_{x in D_k}[z_sty(x)])
- p_c^j := E_{(x, y=c) in D_j}[z_sem(x)]
- tau_nce = 0.3 (Plan A), tau_SCPR = 0.3 (from SAS)

### Complexity Budget

- Reused: backbone / heads / classifier / BN / orth decouple / style bank / class-proto bank
- New: SCPR attention + self-mask + renorm (~30 LOC)
- **0 new trainable, 0 new loss terms, 1 new hyperparameter (tau_SCPR)**

### Core Mechanism (single canonical formula)

g_kj = cos(s_k, s_j)
raw_j = exp(g_kj / tau_SCPR) * 1[j != k]
A_k^c = { j : j != k, p_c^j exists }
w_{k->j}^c = raw_j / sum_{l in A_k^c} raw_l   if j in A_k^c else 0

L_SCPR(z, c) = - sum_{j in A_k^c} w_{k->j}^c * log [
    exp(sim(z, p_c^j)/tau_nce)
    / ( exp(sim(z, p_c^j)/tau_nce)
        + sum_{c' != c, l in A_k^{c'}} exp(sim(z, p_{c'}^l)/tau_nce) )
]

All p_c^j, s_j detached.

Total loss: L = L_CE + 1.0 * L_orth + 0.1 * L_HSIC + 1.0 * L_SCPR (Plan A weights).

### Mathematical Property (single claim)

tau_SCPR -> infinity: w = 1/|A_k^c| -> SCPR strictly reduces to M3 (verified PACS +5.09% lower bound).

### Formal Derivation via entropy-regularized noise minimization (NEW in Round 4)

**Objective**: minimize expected style-noise pulling over weight distributions with entropy regularization.

min_{w in Delta^{K-1}}  J(w) = sum_j w_j * l_j + tau * sum_j w_j log w_j
subject to: sum_j w_j = 1, w_j >= 0

where:
- l_j = E[sim(z, eps_j) | client k] is the style-noise projection expectation from prototype j
- eps_j = p_c^j - p_c^* is the j-domain style residue (nonzero under soft orthogonal decoupling; residual cos(z_sem, z_sty) measured ~0.05-0.1)
- tau > 0 is entropy regularization strength

**First-order condition (Lagrangian)**:
∂J/∂w_j = l_j + tau * (log w_j + 1) + lambda = 0
=> w_j* = exp(-l_j / tau) / Z,   Z = sum_l exp(-l_l / tau)

**Linear approximation** (supported by imperfect decoupling):
l_j ≈ c * (1 - cos(s_k, s_j)) = c * style_dist(k, j),  c > 0

**Substitution**:
w_j* ∝ exp(c * cos(s_k, s_j) / tau) = softmax_j(cos(s_k, s_j) / tau_SCPR)
where tau_SCPR := tau / c

**Conclusion**: SCPR's softmax-over-cosine weighting is the **unique Boltzmann minimizer** of the entropy-regularized noise objective under linear style-dependent noise. It is **derived**, not designed.

**Two limits**:
- Perfect decouple (l_j == 0): J(w) reduces to pure entropy; minimizer is uniform w -> SCPR = M3 exactly.
- Imperfect decouple: style weighting is the optimal noise suppression.
