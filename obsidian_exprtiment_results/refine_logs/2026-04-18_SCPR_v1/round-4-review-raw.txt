## Problem Anchor (preserved)

Replace M3's equal-weighted multi-positive InfoNCE positives with style-conditioned weights, self-masked (j != k). Motivation: **necessary noise-suppression correction from imperfect decouple** (see minimal derivation).

## Proposed Method

### Notation

- s_k := normalize(E_{x in D_k}[z_sty(x)])
- p_c^j := E_{(x, y=c) in D_j}[z_sem(x)]
- tau_nce = 0.3, tau_SCPR = 0.3

### Complexity Budget

- 0 new trainable, 0 new loss terms, 1 new hyperparameter (tau_SCPR)

### Core Mechanism (single canonical formula)

g_kj = cos(s_k, s_j)
raw_j = exp(g_kj / tau_SCPR) * 1[j != k]
A_k^c = {j: j != k, p_c^j exists}
w_{k->j}^c = raw_j / sum_{l in A_k^c} raw_l    if j in A_k^c else 0

L_SCPR(z, c) = -sum_{j in A_k^c} w_{k->j}^c * log [ exp(sim(z, p_c^j)/tau_nce) / (numerator + sum_{c'!=c, l in A_k^{c'}} exp(sim(z, p_{c'}^l)/tau_nce)) ]

All p_c^j and s_j detached.

Total: L = L_CE + 1.0 * L_orth + 0.1 * L_HSIC + 1.0 * L_SCPR (Plan A weights).

### Mathematical Property (single claim)

tau_SCPR -> infinity: w = 1/|A_k^c| -> SCPR strictly reduces to M3 (verified +5.09% on PACS).

### Why style weighting is a necessary correction — Minimal Derivation under imperfect decoupling

Decompose each prototype as `p_c^j = p_c^* + eps_j`, where:
- `p_c^*` is the ideal style-free class-c prototype,
- `eps_j` is j-domain's style residue absorbed through imperfect orthogonal decoupling (cos^2(z_sem, z_sty) → 0 is a soft constraint; residual cos is ~0.05-0.1 in our measurements).

Under multi-positive SupCon on client k, each positive's pulling term decomposes:
  sim(z, p_c^j) = sim(z, p_c^*) + sim(z, eps_j)
                    ——————————      ——————————
                    semantic signal    style-noise pulling (undesired)

The noise term E[sim(z, eps_j) | client k] scales with style_dist(k, j) = 1 - cos(s_k, s_j): far-style prototypes contain more k-foreign style, inducing larger semantic drift in z.

SCPR's weight w_{k->j}^c ∝ exp(cos(s_k, s_j) / tau) applies exponential decay to far-style positives, directly minimizing the expected noise-pulling:

  E[ sum_j w_{k->j}^c * sim(z, eps_j) ]
    = sum_j w_{k->j}^c * c_j * style_dist(k, j)    (c_j > 0)

This expectation is minimized (by standard softmax attention property) when weights follow cos(s_k, s_j).

Two limits:
- Perfect decouple (eps_j = 0 for all j): noise is zero, style weighting is redundant -> uniform w is optimal -> SCPR reduces to M3 seamlessly.
- Imperfect decouple (practical regime): noise is non-zero, style weighting is the minimal single-parameter correction.

**SCPR is not arbitrary reweighting.** It is the necessary noise-suppression mechanism under imperfect decouple. It **can be interpreted as a bias-control / SNR-aware weighting** under imperfect decoupling.
