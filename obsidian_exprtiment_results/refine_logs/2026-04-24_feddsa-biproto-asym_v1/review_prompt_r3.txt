## Thesis (v3, single sentence)

**Domain should be modeled as a first-class federated prototype object Pd, excluded in low-dimensional prototype space against online class centroids derived from the semantic encoder — yielding cross-domain semantic consistency without feature-level adversarial training.**

## Contribution Focus (v3)

- **Dominant (C1, ONLY headline)**: Federated Domain Prototype Pd ∈ ℝ^{D×d_z} with prototype-space exclusion against online class centroids via hybrid straight-through axis. First to model domain as a first-class federated object (Pd aggregated server-side across participating clients, used as the forward anchor of exclusion; gradient flows via batch centroid to encoder_sty).
- **Enabling infrastructure (not contribution)**: Asymmetric statistic encoder_sty (~1M MLP on (μ,σ) taps). Structurally biased against encoding class information.
- **Monitor only (not contribution)**: Pc = EMA class centroid, used for Vis-C plot and sparse-batch fallback.
- **Non-contributions**: not MI-optimal, not communication efficient, no convergence, no cross-backbone, no DP, not trunk decontamination, not dual bank.

## Method (v3)

### Interface Specification

Statistic encoder: Input (μ, σ) from conv1-3 BN-post pre-ReLU taps (detached). Architecture: Linear(in, 512) → LayerNorm → ReLU → Linear(512, 128). Output z_sty L2-normalized.

z_sem source: semantic_head output (L2-normalized 128d), same representation fed to sem_classifier.

Pd EMA update (server, domain-indexed, handles partial participation):
```
for d in range(D):
    participating = {k : client_k participated AND domain(k) == d}
    if participating:
        agg_d = weighted_mean(client_means of participating)
        Pd[d] = F.normalize(0.9 * Pd[d] + 0.1 * F.normalize(agg_d))
    # else: no-op
```

Pc EMA update: identical structure, class-indexed. Used for monitoring + fallback, never as a loss target.

L_proto_excl computation (client-side per batch):
```
present_classes = {c : count_in_batch(c) >= 2}
present_domains = {d : count_in_batch(d) >= 2}  # usually 1 (single-client = single domain)

# Class axis for all C classes
for c in all_classes:
    if c in present_classes:
        class_axis[c] = F.normalize(mean z_sem_i where y_i==c)  # has grad
    else:
        class_axis[c] = Pc[c].detach()  # EMA fallback, no grad

# Domain axis via hybrid straight-through
for d in present_domains:
    bc_d = F.normalize(mean z_sty_i where domain_i==d)  # has grad
    domain_axis[d] = Pd[d].detach() + bc_d - bc_d.detach()
    # Forward == Pd[d], gradient flows through bc_d to encoder_sty

# Exclusion
L_proto_excl = mean_{c, d} cos²(class_axis[c], domain_axis[d])
```
