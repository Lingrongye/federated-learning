## 三、实验运行

### 3.1 统一实验设置

所有实验使用以下统一设置，确保公平对比：

```
骨干网络：ResNet-18（ImageNet预训练）
BN处理：所有方法使用local BN
数据集划分：每个域=1个客户端
FL轮次：PACS 200轮, Digit-5 100轮
本地训练epochs：5
Batch size：32
学习率：0.01 (SGD, momentum=0.9)
随机种子：42, 123, 456（跑3次取均值±标准差）
评估间隔：每5轮评估一次
```

### 3.2 M0: Sanity Check（最先跑这个！）

**目的**：验证FedDSA代码能正常运行，损失下降，无报错。

```bash
cd PFLlib/system

# 用PACS跑10轮快速验证
python main.py \
    -data PACS \
    -m resnet18 \
    -algo FedDSA \
    -nc 4 \
    -gr 10 \
    -ls 5 \
    -lr 0.01 \
    -lbs 32 \
    -eg 2 \
    -t 1 \
    -lo 1.0 \
    -lh 0.1 \
    -ls2 1.0 \
    -tau 0.1 \
    -wr 5 \
    -did 0

# -did 0 表示使用GPU 0
# 预期：10轮内损失下降，无报错，打印出准确率
```

**检查点**：
- [ ] 没有报错（特别注意维度不匹配、HSIC NaN等）
- [ ] 损失在下降
- [ ] 每次evaluate打印的准确率 > 随机水平（>14% for 7类PACS）
- [ ] 风格仓库大小在增长

如果sanity check失败，先debug再继续。

### 3.3 M1: 基线复现

**PFLlib内置基线**（最简单，只换 `-algo` 参数）：

```bash
# FedBN
python main.py -data PACS -m resnet18 -algo FedBN -nc 4 -gr 200 -ls 5 -lr 0.01 -lbs 32 -eg 5 -t 3 -did 0

# FedProto
python main.py -data PACS -m resnet18 -algo FedProto -nc 4 -gr 200 -ls 5 -lr 0.01 -lbs 32 -eg 5 -t 3 -lam 1.0 -did 0

# MOON
python main.py -data PACS -m resnet18 -algo MOON -nc 4 -gr 200 -ls 5 -lr 0.01 -lbs 32 -eg 5 -t 3 -did 0
```

**外部代码库基线**（需要分别去各自仓库跑）：

### 3.5 M3: 核心消融实验

需要修改代码或加参数来关闭/开启特定模块：

```bash
# (a) Full method（同M2）

# (b) Decouple-only（关闭风格仓库）
# 设置 -wr 9999 让warmup永远不结束，等于不做风格增强
python main.py -data PACS -m resnet18 -algo FedDSA -nc 4 -gr 200 -ls 5 -lr 0.01 -lbs 32 -eg 5 -t 3 -lo 1.0 -lh 0.1 -ls2 1.0 -tau 0.1 -wr 9999 -did 0

# (c) Share-only（关闭解耦约束）
# 设置 -lo 0 -lh 0 关闭解耦损失
python main.py -data PACS -m resnet18 -algo FedDSA -nc 4 -gr 200 -ls 5 -lr 0.01 -lbs 32 -eg 5 -t 3 -lo 0 -lh 0 -ls2 1.0 -tau 0.1 -wr 10 -did 0

# (d) Neither（关闭解耦+关闭风格仓库）
python main.py -data PACS -m resnet18 -algo FedDSA -nc 4 -gr 200 -ls 5 -lr 0.01 -lbs 32 -eg 5 -t 3 -lo 0 -lh 0 -ls2 1.0 -tau 0.1 -wr 9999 -did 0

# (e) 约束消融：orth-only
python main.py -data PACS -m resnet18 -algo FedDSA -nc 4 -gr 200 -ls 5 -lr 0.01 -lbs 32 -eg 5 -t 3 -lo 1.0 -lh 0 -ls2 1.0 -tau 0.1 -wr 10 -did 0

# (f) 约束消融：HSIC-only
python main.py -data PACS -m resnet18 -algo FedDSA -nc 4 -gr 200 -ls 5 -lr 0.01 -lbs 32 -eg 5 -t 3 -lo 0 -lh 0.1 -ls2 1.0 -tau 0.1 -wr 10 -did 0
```


### 5.2 读取结果

```python
import h5py
import numpy as np

# 读取某次实验结果
with h5py.File('results/PACS_FedDSA_test_1.h5', 'r') as f:
    test_acc = np.array(f['rs_test_acc'])
    train_loss = np.array(f['rs_train_loss'])

print(f"Best accuracy: {max(test_acc):.4f}")
print(f"Final accuracy: {test_acc[-1]:.4f}")
```

### 5.3 生成对比表格

```python
import h5py, numpy as np, os

methods = ['FedAvg', 'FedProx', 'FedBN', 'FedProto', 'FedDSA']
dataset = 'PACS'
seeds = 3

for method in methods:
    accs = []
    for t in range(1, seeds + 1):
        path = f'results/{dataset}_{method}_test_{t}.h5'
        if os.path.exists(path):
            with h5py.File(path, 'r') as f:
                accs.append(max(np.array(f['rs_test_acc'])))
    if accs:
        print(f"{method}: {np.mean(accs)*100:.2f} ± {np.std(accs)*100:.2f}")
```

## 七、常见问题排查

| 问题 | 原因 | 解决 |
|------|------|------|
| `CUDA out of memory` | batch size太大或模型太大 | 减小 `-lbs` 到16或8 |
| `FileNotFoundError: .npz` | 数据集未生成 | 先运行 `generate_PACS.py` |
| HSIC输出NaN | batch太小或特征全相同 | 增大batch size到32+ |
| 准确率始终~14%不提升 | 学习率太大或模型初始化问题 | 减小lr到0.005，检查模型加载 |
| `ModuleNotFoundError` | 依赖未安装 | `pip install xxx` |
| 风格仓库始终为空 | warmup期间不收集/client未上传 | 检查warmup_rounds设置 |
| 基线结果与论文差距大 | 实验设置不同（骨干、预训练、数据划分） | 对齐设置或在论文中注明 |
